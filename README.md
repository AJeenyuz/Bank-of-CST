# Bank-of-CST

Project: Banking Database - Loading Relations

This dataset was manually created based on the schema designed in Part 2 of the project.

Source of Data:
- Customer, Account, Transactions, and Login data were created by the project group.
- No external datasets were used.

File Descriptions:
- create.sql: Contains SQL statements to create all tables
- load.sql: Contains SQL statements to load data using LOAD DATA
- customer.dat: Customer data
- account.dat: Account data
- transactions.dat: Transaction data
- login.dat: Login credential data

Notes:
- Files are in dat format (comma-separated) with header rows
- Dates are formatted as YYYY-MM-DD
- NULL values are used where appropriate (e.g., withdrawals without a destination account)
